## 素材优化管理系统
### (一) 智能测试模型
```ad-rotation
├─ 图片类型
│  ├─ 尺寸规格：1200x628 → 动态适配
│  └─ 创意组合：场景化展示+痛点放大
└─ 视频类型
   ├─ 时长梯度：6秒/15秒/30秒
   └─ 元素组合：产品特写+用户证言
```

### (二) 素材迭代机制
| 状态阶段 | 核心指标            | 优化动作                 |
|----------|---------------------|--------------------------|
| 导入期   | CTR<1%             | 文案AB测试+版位调整      |
| 成长期   | CPM波动>20%        | 时段分层+预算再分配       |
| 衰退期   | ROAS降幅>25%       | 素材版本全量更新          |

## 动态预算调控法则
### (一) 分时优化公式
```math
预算系数 = \frac{历史转化率}{时均竞争指数} × \sqrt{素材新鲜度}
```

### (二) 异常处理流程
```flow
st=>start: CPM突增30%
op1=>operation: 暂停超标广告组
op2=>operation: 启动替代广告方案
cond=>condition: CTR降幅≤15%?
op3=>operation: 时段拆分优化
op4=>operation: 全量素材替换
e=>end: 数据恢复基准值

st->op1->op2->cond
cond(yes)->op3->e
cond(no)->op4->e
```
